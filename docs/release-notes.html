HanYoungHater 1.0 릴리즈 노트

출시일: 2024년 9월 18일

안녕하세요, HanYoungHater의 첫 번째 공식 릴리즈를 소개합니다! 이번 버전은 맥OS 사용자들에게 특별한 경험을 선사하기 위해 제작되었으며, 아래와 같은 주요 기능과 개선 사항들이 포함되어 있습니다.

주요 기능

  • 사용자 맞춤형 UI: 간결하고 직관적인 SwiftUI 기반 인터페이스로 누구나 쉽게 사용할 수 있습니다.
  • 자동 업데이트 지원: Sparkle을 통해 최신 버전으로 자동 업데이트가 가능하여, 새로운 기능을 놓치지 않고 사용할 수 있습니다.
  • 안정성과 성능 최적화: 앱의 전반적인 성능을 향상시키고, 빠르고 안정적인 환경을 제공합니다.

부가 기능

  • 다양한 시스템 호환성: macOS 13.0 Ventura 이상에서 원활하게 작동하도록 최적화되었습니다.
  • 향상된 반응 속도: 사용자 인터페이스와 데이터 처리가 더욱 빠르게 작동하도록 성능을 개선했습니다.
  • 자동 업데이트
  • 로그인 시 자동실행

앞으로의 계획

이 릴리즈는 HanYoungHater의 첫 걸음일 뿐입니다. 앞으로 더 많은 기능과 업데이트가 예정되어 있으니 기대해 주세요!
